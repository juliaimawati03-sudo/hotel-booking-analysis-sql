Hotel Booking Analysis SQL
1. Project Overview
This project analyzes hotel booking data using SQL to uncover booking trends, customer behavior, cancellation patterns and business insights.

2.Tools Used
- MySQL
- SQL
- CSV Dataset

3.Business Questions
1. What is the cancellation rate?
2. Which hotel type receives the most bookings?
3. How does lead time affect cancellations?
4. Which countries generate the most bookings?
5. What are the monthly booking trends?

4.Key Insights
- City Hotel has the highest number of bookings.
- Long lead times tend to have higher cancellation rates.
- Seasonal trends significantly affect booking volume.
